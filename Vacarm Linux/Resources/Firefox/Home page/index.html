Welcome to Vacarm Linux 0.1 !

Vacarm Linux is a security-oriented GNU/Linux distribution. It includes the 40 most famous tools to carry out audits, forensic analyses, network monitoring, intrusion detection, etc.
The distribution is designed to help people having a secure and user-friendly operating system.

Getting Help with Vacarm Linux :

Access Vacarm Linux's documentation on the Vacarm Linux's website. You can also find help from the official Ubuntu documentation website and the community contributed documentation.

Participate in Vacarm Linux :

You want to contribute to Vacarm Linux project !
We invite you to send us a mail to specify the nature of your contribution. We accept any kind of help : money, distribution hosting, developpement, etc.
To find out how to participate in Ubuntu, visit Ubuntu's community participation page!
